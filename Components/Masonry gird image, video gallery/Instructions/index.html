How To use

Open the 'index.html' file with a text editor, you can add more images by copying one of the 'item' elements and placing your own link to your image and the title in the attributes

For example, below in the 'src' attribute add the source to the image and in the 'data-title' we add the title of the image

Example for images

For videos, you would folow the exact same steps as previously, except now the in the 'src' we specify the source for the video thumbnail, which can be any image. We also specify the title in the 'data-title' as before. And the source of the video should be entered in the 'data-source' attribute
